Este é um projeto Android desenvolvido em Kotlin que explora a navegação entre telas de um aplicativo. Primeiro veio a LoginScreen a porta de entrada do app, com um layout simples e um botão para o usuário acessar o sistema. Logo depois surgiu a MenuScreen, oferecendo três caminhos ao usuário: ir para o Perfil, ver os Pedidos ou Sair. Em seguida foram criadas as telas de destino a PerfilScreen, com um botão para voltar ao menu, e a PedidosScreen, com seu layout básico e título.Com as telas prontas conectei tudo. A LoginScreen foi definida como tela inicial do app diretamente na MainActivity. O passo seguinte foi implementar o NavController na MainActivity, o componente responsável por controlar qual tela está ativa em cada momento. E por fim a navegação passou a funcionar de forma completa, com o NavController e o innerPadding sendo repassados corretamente para cada tela através da MainActivity.
